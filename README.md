# Stock Hogar

Aplicación web para administrar stock de productos por hogares/casas.

Permite:
- Gestión de usuarios.
- Gestión de hogares.
- Visualización de productos.
- Control de stock por casa.
- Manejo de lotes con fecha de vencimiento.
- Ingreso y egreso de stock.
- Edición y eliminación de lotes.
- Alertas de stock mínimo.

---

# Requisitos previos

Antes de instalar el proyecto se necesita tener instalado:

- Node.js (versión recomendada: 18 o superior)
- npm

Verificar instalación:

En la consola Bash
node -v
npm -v

Instalación Backend:
Ingresar a la carpeta del backend

En la consola Bash
cd backend

Instalar dependencias:
npm install

Ejecutar Backend
Levantar servidor:
npm start



Instalación Frontend
Abrir otra terminal.

Ingresar:
cd frontend

Instalar dependencias:
npm install

Ejecutar Frontend:
npm run dev
